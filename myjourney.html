My Journey

ที่ผมสามารถเข้ามาเรียนใน CSTU ได้ ทำให้ผมดีใจเป็นมาก เพราะเป็นคณะ สาขา และมหาวิทยาลัยที่ผมอยากจะเข้ามาตั้งแต่มัธยมต้นแล้ว และการที่ผมได้เข้ามาเรียน ก็เพื่อที่จะมาพัฒนาศักยภาพของตัวผมเองในอนาคต ในอีก 4 ปีนี้ สิ่งที่ผมต้องการจะได้จากการเรียนมีหลายอย่างด้วยกัน ยกตัวอย่าง เช่น การเขียนเว็บ เกี่ยวกับ UI/UX ภาษาที่หลากหลายเช่น html css c java และกระทั่งเรื่อง network แล้วหลังจากที่ได้เรียนก็อยากจะนำไปต่อยอดเองต่อไป
