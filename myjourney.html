My Journey

การที่ผมสามารถเข้ามาเรียนใน CSTU ได้นั้น ทำให้ผมดีใจเป็นอย่างมาก เพราะเป็นคณะ สาขา และมหาวิทยาลัยที่ผมอยากจะเข้ามาตั้งแต่มัธยมต้นแล้ว และการที่ผมได้เข้ามาเรียนก็เพื่อที่จะมาพัฒนาศักยภาพของตัวผมเองในอีก 4 ปี ในระยะเวลานี้ สิ่งที่ผมต้องการจะได้จากการเรียนมีคร่าวๆ เช่น การเขียนเว็บ เกี่ยวกับ UI/UX ภาษาที่หลากหลายเช่น html css c java หรือแม้กระทั่งเรียนเรื่อง network และหลังจากที่ได้เรียนก็อยากจะนำไปต่อยอดเองต่อไปครับ
